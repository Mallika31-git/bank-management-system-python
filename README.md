# 🏦 Bank Management System

A console-based Bank Management System built using Python.

## Features

- Create Account
- Generate Account Number
- Deposit Money
- Withdraw Money
- Transfer Money
- Balance Enquiry
- Transaction History
- Delete Account
- Display Accounts

## Concepts Used

- Python Functions
- Dictionaries
- Lists
- Loops
- Conditional Statements

## How to Run

```bash
python bank_management_system.py
```
